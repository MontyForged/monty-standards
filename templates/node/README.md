# Node.js Template

Copy these files to your Node.js project:

```
.
├── package.json
├── vitest.config.js
├── .eslintrc.json
├── src/
│   └── index.js
└── tests/
    └── example.test.js
```

## Usage

```bash
# Install dependencies
npm install

# Run tests
npm test

# Run linter
npm run lint

# Format code
npm run format

# Full check
npm run check
```
